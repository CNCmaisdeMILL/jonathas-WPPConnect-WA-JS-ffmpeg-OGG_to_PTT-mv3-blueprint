# jonathas-WPPConnect-WA-JS-ffmpeg-OGG_to_PTT-mv3-blueprint
Motor Stealth de áudio para WhatsApp Web (PTT) em Chrome MV3. Converte áudios via FFmpeg WASM (Offscreen) com spoofing de metadados (48kHz, Recorder) para mimetizar notas de voz nativas. Blueprint completo com integração WPPConnect/WA-JS e gerenciamento de fila via Supabase para automações profissionais.
